
# Kaptan Bulut: Güneşli Günler ☀️☁️

Çocuklar için eğlenceli ve motive edici bir kuruluk takibi uygulaması. Kaptan Bulut ile ıslak geceleri geride bırakın!

## Özellikler
- **Günlük Takip:** Gece kuruluk durumunu neşeli bir arayüzle kaydedin.
- **Yapay Zeka Raporu:** Gemini AI ile verilerinizi analiz edin ve hekiminiz için klinik özetler oluşturun.
- **Mesane Günlüğü:** 24 saatlik detaylı sıvı alımı ve idrar çıkışı takibi.
- **Madalya Sistemi:** Başarılarınızı yıldızlar ve özel madalyalarla kutlayın.
- **Kaptan'ın Rehberi:** Uzman görgörüşleriyleüşleriyle hazırlanan eğitici içerikler.

## Kurulum
Bu proje Vite ve React ile geliştirilmiştir.

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```
